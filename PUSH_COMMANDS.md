# GitHub Push Commands

After creating your GitHub repository, run these commands in PowerShell:

```bash
cd Event_Management_System_For_Colleges

# Add your GitHub repository as remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/Event-Management-System-For-Colleges.git

# Push to GitHub
git push -u origin master
```

Replace YOUR_USERNAME with your actual GitHub username.

Example:
git remote add origin https://github.com/johndoe/Event-Management-System-For-Colleges.git
git push -u origin master
```
